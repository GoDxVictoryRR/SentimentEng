Usage: puter app [options] [command]

App management commands

Options:
  -h, --help                  display help for command

Commands:
  info <app_name>             Get application information
  create <name> <remote_dir>  Create a new app
  update <name> [dir]         Update an app
  delete [options] <name>     Delete an app
  help [command]              display help for command
