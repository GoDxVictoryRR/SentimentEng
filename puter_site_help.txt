Usage: puter site [options] [command]

Site management commands

Options:
  -h, --help                         display help for command

Commands:
  info <site_uid>                    Get site information by UID
  create [options] <app_name> [dir]  Create a static website from directory
  deploy [local_dir] [subdomain]     Deploy a local web project to Puter
  delete [options] <uid>             Delete a site by UID
  help [command]                     display help for command
